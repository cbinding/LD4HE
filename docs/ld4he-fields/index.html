OASIS mandatory fields

The source data for the LD4HE project comes from a specific subset of fields in the overall OASIS dataset. These are mandatory fields in the OASIS data input.

Field ID Field Name Notes
1 oasis_id Unique identifier of the OASIS record)
2 event_type The type of investigation. URIs from FISH Event Types Thesaurus. There may be multiple event types per record
3 reason for investigation The reason for undertaking an investigation. URIs from OASIS Reason for Investigation
4 Country Location of a site. URIs from OS Open Names
5 Site name Free text name for a local site
9a Grid reference (geom_ngr) Location of a site. This will store either a point, line or polygon in a single geometry field (using PostGIS), using the OSGB36 crs. There can be multiple values per record.
9b Grid reference (geom_ll) Location of a site. This will store either a point, line or polygon in a single geometry field (using PostGIS), using the WGS84 crs. There can be multiple rows per record.
15 County Location of a site. URIs from OS Open Names
16 District Location of a site. URIs from OS Open Names
17 Parish Location of a site. URIs from OS Open Names
18 HER Auto generated. HER organization responsible for the area encompassing the location of the site
19 National body Auto generated. National body responsible for the area encompassing the location of the site
22 Project title Value generated in form
23a Start date Start of the overall timespan for an investigation. Value generated in form
23b End date End of the overall timespan for an investigation. Value generated in form
24a Methodology Techniques used in an investigation, URIs from FISH Event Types. There can be multiple values per record.
24b Description Free text
28 Planning application ID Free text, and only if prompt (id3) = commercial
31 Project codes Type of code - OASIS Associated ID
35 Publication type URIs from OASIS Paper and Digital Archive Component vocabulary
91 Type of report Report type. URIs from OASIS Paper and Digital Archive Component vocabulary
36 Title of report Report title. Auto generated through form
39 Author / editor Author / Editor of report. Potential at a future date to assign ISSN to authors via the ADS Library, but the methodology for sustaining this is still not firmly established.
44 Date year the report was published/issued
45 Publisher Organization publishing the report
46 Place of issue Place of publication for the report. Could use OS, but of limited use for this data
58 Name of organization Auto generated
50 URL URL of the report
50a DOI DOI of the report - an identifier
62 Monument type URIs from monument type vocabulary corresponding to site location: England / Scotland / Wales
63 Monument period URIs from period vocabulary corresponding to site location: England / Scotland / Wales
64 Artefact type URIs from object types vocabulary according to site location: England & Wales / Scotland
65 Artefact period URIs from period vocabulary corresponding to site location: England / Scotland / Wales
70 Research outcomes Use selection from Research Frameworks list (non-LOD at time of writing)
OASIS+ Solid geology If Geophysical module completed. Will use data from http://data.bgs.ac.uk/downloads.html
OASIS+ Drift geology If Geophysical module completed. Will use data from http://data.bgs.ac.uk/downloads.html
